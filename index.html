Visual Odometry (SfM / SLAM)

The Project was about estimating the camera pose by using just the sequence of images (i.e. a video) and plot the path in the world co-ordinate frame. We implemented the epipolar geometry concepts from scratch and then compared the result with the built in cv2 functions (an open source Computer Vision library).

Traffic Sign Detection and Classification for Self-Driving Cars

This project’s objective is to detect traffic sign from a video footage taken from a car in motion and classify these detected signs. So detectd the reigon of interest (traffic signs) using Contrast Stretching, Intensity Normalization and Maximum Stable Extremal Regions (MSER). Extracted HOG features of the detected traffic signs and used SVM for their classification.
